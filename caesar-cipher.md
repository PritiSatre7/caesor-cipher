## Caesar Cipher

### Description
Caesar Cipher is a basic encryption technique where each letter in the text is shifted by a fixed number of positions in the alphabet.
It is used to convert readable text into encrypted text and can be reversed using the same shift value for decryption..

### Encryption Example
Input: HELLO  
Shift: 2 
Output: JGNNQ

### Decryption Example
Input: JGNNQ  
Shift: 2 
Output: HELLO
